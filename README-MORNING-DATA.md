# Morning Data Integration — v0.5.1

The Morning collection is now data-driven.

## Data source

`js/data/morning-data.js`

This browser-ready file contains all 16 extracted Morning entries and works when the site is opened directly from the computer.

## Rendering

`js/morning-collection.js` generates every card from the structured data.

## Visibility

Verification metadata is preserved but does not hide entries in this development version.
