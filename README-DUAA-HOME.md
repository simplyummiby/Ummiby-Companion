# Duaa Home — v0.4.0

This full-site package preserves the shared Home and Qur’an Home while adding the Duaa Home dashboard at:

`duaa/index.html`

## Included

- Daily Companion cards for Morning, Evening, and Before Sleep
- Gentle progress overview
- Collection shortcuts
- Recently Viewed
- Functional local reminder preference
- Pause and resume reminder controls
- Responsive layouts
- Placeholder destination pages to prevent broken navigation

## Reminder behavior

Reminder preferences are stored in local storage for this prototype.

Actual browser or device notifications are not yet implemented.

## Install

Replace the repository contents with this complete package, or carefully merge it into the repository.

The full package is provided to prevent partial-file overwrites from breaking the root `index.html`.
