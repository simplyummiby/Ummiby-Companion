# Update Instructions — Ummiby Companion v0.7.3.1

This package updates Ummiby Companion v0.7.3 to v0.7.3.1.

## Recommended update method

1. Back up any local-only changes.
2. Extract this ZIP.
3. Copy the contents into the root of your existing Ummiby Companion repository.
4. Allow matching files to be replaced.
5. Open `quran/index.html` and confirm the new **Memorization** card is visible.
6. Open the Qur’an sidebar and confirm **Memorization** is listed in the navigation.

## What changed in v0.7.3.1

- Added a prominent Memorization entry card to Qur’an Home.
- Added Memorization access to Qur’an navigation menus throughout the module.
- Preserved the existing Memorization landing page.
- Updated release-wide version references.

No memorization progress data is introduced in this release.
