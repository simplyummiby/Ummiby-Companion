# Update Instructions — Ummiby Companion v0.7.4

This package updates Ummiby Companion v0.7.3.1 to v0.7.4.

## Recommended update method

Replace the existing project files with the contents of this ZIP, preserving any deployment-specific files that are not part of the repository.

## What changed in v0.7.4

- Added the complete 114-surah Memorization Library.
- Added elegant responsive surah cards.
- Added search by surah name, Arabic name, or number.
- Added hifdh-status filter chips.
- Added Makki and Madani filters.
- Added a working memorization detail route for every surah.
- Updated the Memorization roadmap and release documentation.
