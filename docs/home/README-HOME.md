# Shared Home — v0.2.0

This package contains the root Home page for the combined Quran and Duaa application.

## Scope

Included:

- Root `index.html`
- Shared sidebar navigation
- Quran feature panel
- Duaa feature panel
- Collapsed “What are Reading Units?” accordion
- Responsive desktop, tablet, and mobile behavior
- Temporary decorative non-living SVG artwork

Not included:

- Quran Home/dashboard
- Duaa Home/dashboard
- Settings page functionality
- Quran or Duaa content datasets
- Progress or tracking logic

## Install

Copy the package contents into the root of the repository.

The links intentionally point toward future locations such as:

- `quran/index.html`
- `duaa/index.html`

Those pages will be built separately.
