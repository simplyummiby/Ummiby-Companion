# Shared Home Page

## Purpose

The root `index.html` introduces the two main areas of the application:

- Qur’an
- Duaa

It is an orientation page, not a personal dashboard.

## Main Features

### Qur’an

- Reading Journey
- Ramadan Reading Journey
- Read by Surah
- Go to an Ayah
- “What are Reading Units?” explanation

### Duaa

- Daily Companion
- Morning, Evening, and Before Sleep tracking
- Find a Duaa
- Recitation and Focus Mode paths

## Boundaries

The shared Home does not show personal progress. Progress belongs on the separate Qur’an Home and Duaa Home dashboards.
