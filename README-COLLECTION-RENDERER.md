# Reusable Duaa Collection Renderer — v0.5.2

## Main page

`duaa/collection.html`

## URL examples

- `duaa/collection.html?collection=morning`
- `duaa/collection.html?collection=evening`
- `duaa/collection.html?collection=sleep`
- `duaa/collection.html?collection=travel`
- `duaa/collection.html?collection=weather`
- `duaa/collection.html?collection=prayer`

## Data

`js/data/duaa-collections.js`

## Result

Adding a new duaa to a collection’s `items` array automatically creates a new card when the page reloads.

Tracking is enabled only when the collection metadata says `trackerEnabled: true`.
